## Key Features Added

### Extraction (loop-vault)
- 5% fee on extraction
- Resets Cred balance to zero
- Liquidates all stacking positions
- OXO unaffected

### veOXO Decay (loop-oxo)
- Linear decay from initial veOXO to 0 at unlock
- `calculate_decayed_ve_oxo()` helper function
- Fee share uses decayed balance

### Agent Identity (loop-avp)
- Personal Agent: bound to principal via hash
- Service Agent: requires 500 OXO stake
- Capability declarations (up to 20)
- Reputation scoring (0-10000 basis points)
- Suspend/revoke/reactivate lifecycle
