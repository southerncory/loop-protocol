# The Scale of Extraction

Understanding the magnitude of value extraction requires looking at both individual impact and systemic flows.

## Individual Extraction Analysis

For a US consumer with median income and typical spending patterns:

### Direct Transaction Costs

Annual spending of $50,000 generates:

| Fee Type | Rate | Annual Cost |
|----------|------|-------------|
| Credit card interchange | 2.3% average | $1,150 |
| Debit card fees | 0.5% average | $100 |
| ATM and bank fees | Flat | $60 |
| Foreign transaction fees | 3% of $1,500 | $45 |
| **Transaction subtotal** | | **$1,355** |

### Data Value

Your behavioral data has measurable market value:

| Data Type | Market Value |
|-----------|--------------|
| Purchase history | $30-50/year |
| Location data | $50-80/year |
| Browsing behavior | $100-150/year |
| Social connections | $40-60/year |
| App usage patterns | $30-50/year |
| **Data subtotal** | **$250-390/year** |

### Attention Economics

Average daily screen time in the US is over 7 hours. Advertising revenue per user varies by platform:

| Platform Type | Annual Revenue Per User |
|---------------|-------------------------|
| Social media | $150-250 |
| Search | $100-150 |
| Video streaming (ad-supported) | $50-100 |
| **Attention subtotal** | **$300-500/year** |

### Total Annual Extraction

| Category | Low Estimate | High Estimate |
|----------|--------------|---------------|
| Transaction fees | $1,355 | $1,355 |
| Data monetization | $250 | $390 |
| Attention value | $300 | $500 |
| **Annual total** | **$1,905** | **$2,245** |

## The Compound Effect

The real cost isn't the annual extraction. It's the compound growth you lose.

If you captured $2,000 annually and invested it at 7% returns:

| Years | Total Contributed | Compounded Value |
|-------|-------------------|------------------|
| 10 | $20,000 | $29,500 |
| 20 | $40,000 | $87,700 |
| 30 | $60,000 | $202,100 |
| 40 | $80,000 | $427,200 |

That's $427,000 in potential wealth from capturing value that currently flows to intermediaries. The extraction itself is significant; the lost compound growth is enormous.

## Systemic Scale

Individual numbers scale to massive systemic flows:

**US Card Interchange**: $100+ billion annually flows from merchants (ultimately consumers) to card networks and banks.

**Global Data Brokerage**: The data broker market exceeds $300 billion annually, built primarily on consumer data collected without meaningful compensation.

**Digital Advertising**: $600+ billion in global digital advertising revenue, generated by user attention, captured entirely by platforms.

These aren't obscure financial instruments. They're the economic flows underlying everyday activities.

## Distribution Asymmetry

The extraction isn't evenly distributed. People with higher transaction volumes and more digital engagement generate more value for extractors while receiving the same nothing in return.

Meanwhile, the entities capturing this value have:

- Sophisticated technical infrastructure refined over decades
- Legal teams that shape regulations in their favor
- Network effects that make competition difficult
- User agreements that most people never read

Individual consumers have no comparable infrastructure for participating in these value flows.

## The Agent Multiplier

As AI agents begin handling more economic activity, extraction will scale:

**More transactions**: Agents will make micro-purchases, subscriptions, and optimizations that humans wouldn't bother with. Each generates extractable fees and data.

**Richer data**: Agents will access more services, make more queries, and generate more behavioral signal than humans currently do.

**Higher velocity**: Agents operate 24/7 without the friction of human attention. Transaction velocity could increase by an order of magnitude.

If current infrastructure remains, agents will simply accelerate extraction. A future where everyone has an AI agent could mean dramatically more value flowing to incumbents.

The alternative is building infrastructure where agents capture value for users instead.
