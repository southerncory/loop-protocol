# Glossary

Key terms used in Loop Protocol documentation.

## A

**Agent**: Software that acts on behalf of a human. Manages vaults, submits proofs, executes transactions.

**AVP (Agent Value Protocol)**: Protocol layer handling agent identity, principal binding, and capability management.

**APY**: Annual Percentage Yield. Loop stacking yields range from 3-15% APY.

**Auto-Stack**: Feature that automatically restacks Cred when positions mature.

## C

**Capture**: Intercepting value from activities and routing it to user vaults.

**Capture Module**: Pluggable component handling value capture for specific activity types.

**Cred**: Stable value token. 1 Cred = $1 USD, backed by USDC.

## E

**Escrow**: Value held by protocol until conditions are met.

**Extraction**: Converting Cred to USDC and withdrawing. 5% fee applies.

## O

**OXO**: Governance and equity token. Fixed supply of 1 billion.

## P

**Personal Agent**: Agent serving one human. Manages their vault.

**Principal**: The human an agent represents.

**Proof**: Cryptographic evidence of an activity, submitted for capture.

## S

**Service Agent**: Agent serving multiple users. Requires stake, can launch tokens.

**Stacking**: Locking Cred to earn yield.

**Stake**: Collateral posted by Service Agents.

## V

**veOXO**: Vote-escrow OXO. Locked OXO providing governance power and fee share.

**VTP (Value Transfer Protocol)**: Protocol layer handling value movement.

**Vault**: User's personal value store on Loop.

## Numbers

**85/10/5**: Default capture split (user/protocol/stakers).

**500 OXO**: Fee to create a Service Agent (burned).

**25,000 OXO**: Graduation threshold for agent token bonding curves.
