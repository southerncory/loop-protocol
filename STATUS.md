## Last Session

**Date:** 2026-03-10

**What happened:**
- Created loop-protocol repository
- Established full directory structure
- Created initial documentation scaffold
- Defined architecture, tokenomics, and all module specs

**Decisions made:**
- Protocol name: Loop Protocol
- Tokens: Cred (stable) + OXO (equity)
- Chain: Solana
- Two agent types: Personal Agents + Service Agents
- Modular capture layers (shopping first, then data, presence, attention)
- Adopted patterns from Virtuals Protocol (bonding curve, veToken, ACP state machine)
- User-centric philosophy (value flows to humans, not agent creators)
